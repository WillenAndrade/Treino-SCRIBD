Crie uma estrutura condicional switch que receba uma string com o nome de uma fruta e que possua três casos:
Caso maçã, retorne no console: “Não vendemos esta fruta aqui”.
Caso kiwi, retorne: “Estamos com escassez de kiwis”. Caso melância, retorne: “Aqui está, são 3 reais o quilo”. Teste com estas três opções . Crie também um default, que retornará uma mensagem de erro no console.
